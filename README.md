# Django Restaurant QR Code Generator Project

This project is a Django web application that generates QR codes for restaurant menus. 
Customers can scan the QR code to view the digital menu.

## Features
- Generate QR codes for menu links
- Save QR images in the media folder
- Display QR results on a webpage
- Download QR code

## Technologies Used
- Python
- Django
- HTML / CSS
- qrcode Python library
